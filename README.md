# 📈 Salary Prediction — Linear Regression

## 📌 Objective
Predict employee salary based on years of experience using Simple Linear Regression.

## 📊 Dataset
- **Source:** Kaggle — Salary Dataset
- **Features:** Years of Experience
- **Target:** Salary (in USD)
- **Size:** 35 records

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## ⚙️ How It Works
1. Load and explore the Salary dataset
2. Split data into training and test sets
3. Train a Linear Regression model using Scikit-learn
4. Predict salaries on test data
5. Evaluate using Mean Squared Error (MSE)

## 📉 Model Results
- **Model:** Linear Regression
- **Evaluation Metric:** Mean Squared Error (MSE)
- **Key insight:** Strong positive correlation between experience and salary

## 💡 What I Learned
- How to implement Simple Linear Regression from scratch using Sklearn
- How to split data into train/test sets
- How to evaluate model performance using MSE
- Visualizing regression lines with Matplotlib

## 🚀 How to Run
1. Open `linearregression_MN2.ipynb` in Google Colab or Jupyter Notebook
2. Upload `Salary.csv` dataset
3. Run all cells
